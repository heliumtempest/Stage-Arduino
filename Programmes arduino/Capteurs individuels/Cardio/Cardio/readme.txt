Ce programme effectue les relevés de la fréquence des battements cardiaques

"Cardio" est la première ligne envoyée par le programme, il s'agit de son nom et le programme python peut alors chercher le
module correspondant au capteur sur la base de ce nom.
Le programme envoie ensuite le fréquence des battement cardiaques et le temps écoulé depuis le lancement du programme. Ces deux
valeurs sont séparées d'un espace.
